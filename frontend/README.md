# FogChess Frontend

Vue 3 + TypeScript + Vite frontend for FogChess (Fog of War Chess).

## Stack

- **Vue 3** – Composition API, `<script setup>` SFCs
- **TypeScript** – type-safe development
- **Vite** – fast dev server and builds
- **Pinia** – state management
- **Vue Router** – routing
- **Socket.io-client** – real-time game communication

## Development

```bash
npm install
npm run dev
```

See [Vue Docs TypeScript Guide](https://vuejs.org/guide/typescript/overview.html#project-setup) for setup details.
